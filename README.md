# MAXOMOD_NULISA
Code for analysis of NULISA proteomics data of MAXOMOD samples.
To Run the pipeline, please use 'bash Run.sh'

00_initialization.R: Data preparing
01_data_mining.R: Correlation plot between fluid; NPQ boxplot for each target; Overall Target Detectability
02_analysis_groups.R: ANOVA and t-test with and without adjustment
03_PCA_vis.R: PCA visualization
